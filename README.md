# Trabalho de Fisica ⛵
O presente trabalho tem como intuito apresentar um programa que solucione um problema de física, apresentando as limitações de INPUT e mostrando os resultados determinados pelos dados de entrada.

Tal como descrito, o problema em si consiste em calcular o tempo de queda de uma chave que está caindo de uma ponte até um barco, e a velocidade desse barco para que a colisão aconteça. 
Todo o projeto foi desenvolvido utilizando a linguagem de programação Java. 

Além disso, foi adicionado um bônus de animação do OpenGL de uma chave caindo em um barquinho, para ilustrar a situação.

## Funcionalidades do Programa 👀

  -Interface Gráfica do Swing;
  
  -Especificações dos dados de entrada(caso o valor inserido não esteja contido dentro dos limites impostos, o programa apresenta uma mensagem ao usuário para corrigir o input);
  
  -Calculo da queda da chave e da velocidade do barco, para casos em que possue velocidade inicial ou não(para a chave), ou se possue aceleração ou não(para o barco);
  
  -Mudança visual do tamanho da ponte e do barco;
